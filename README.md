# NUXT-STARTER-LAYER

A simple Nuxt + Nuxt/content website base layer,
designed to support Tangueros.Net


## Setup

```bash
pnpm install
```

## Development

```bash
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view app.

## Deployment

Runs in any static hosting (github pages) or (vercel, netlify)
